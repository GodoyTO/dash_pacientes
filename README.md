# Dashboard de acompanhamento de Pacientes
Dashboard com ferramentas de acompanhamento da jornada dos pacientes desde a consulta inicial até o monitoramento da evolução clínica e efeitos adversos

# Dados aleatórios
Os dados são totalmente aleatórios derivados de uma [ferramente de geração aleatória de nomes e outras informações](https://www.fakenamegenerator.com/order.php). Com essa ferramenta conseguimos:
Nomes, sobrenomes, sexo, data de nascimento, altura, peso, cidade, UF, longitude e latitude do paciente.
A ferramenta gera também outros valores que podem ser utilizados para criação de variáveis como estágio da doença, comorbidade, achado clínico, diagnóstico, tratamento e afins.
